base¶

All Batch dialogs are created by subclassing from BatchDialogBase or BatchDialogBaseCopy to inherit common batch functionality:

  • choosing font files from a folder
  • creating unique font IDs
  • managing a list of target fonts

Each base object handles a different batch action pattern:

  • BatchDialogBase: doing something to a set of fonts
  • BatchDialogBaseCopy: copying something from one font to a set of fonts
