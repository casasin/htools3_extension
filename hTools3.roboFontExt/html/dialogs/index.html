Dialogs¶

Overview¶

hTools3 dialogs are organized around common patterns or scopes in font production tasks:

  • Glyphs: applying actions to selected glyphs in the current font
  • Font: applying font-level actions to a font
  • Batch: applying an action to several fonts at once
  • Glyph: visualising and transforming single glyphs in the glyph window
  • Proof: generating different kinds of proofs for the current font

Single dialogs are created using object inheritance. All dialogs inherit from the hDialog class, which defines the basic look’n’feel and generic functionality. Batch and Glyphs dialogs inherit from higher-level BatchDialogBase and GlyphsDialogBase classes.

  • Misc: widgets and components for use in hTools3 dialogs.
