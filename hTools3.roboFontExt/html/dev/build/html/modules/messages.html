messages¶

A collection of standard messages for use in hTools3 dialogs.

hTools3.modules.messages.showMessage(messageText, mode=0)¶

Display a given message to the user.

mode

The display mode of the message:

  1. print to console

  2. show banner notification

hTools3.modules.messages.noXSelected(x)¶

Generic message to be used when a required object is not selected.

hTools3.modules.messages.noXOpen(x)¶

Generic message to be used when a required object is not open.

hTools3.modules.messages.noPointSelected = 'No point selected.\nPlease select one or more points.\n'¶

Message to be used when no point is selected.

hTools3.modules.messages.onlyOnePoint = 'Only one point selected.\nPlease select at least two points.\n'¶

Message to be used when only one point is selected.

hTools3.modules.messages.atLeastTwoPoints = 'Please select at least two points.\n'¶

Message to be used when at least two points must be selected.

hTools3.modules.messages.noContourSelected = 'No contour selected.\nPlease select one or more contours.\n'¶

Message to be used when no contour is selected.

hTools3.modules.messages.noGlyphOpen = 'No glyph open.\nPlease open at least one glyph.\n'¶

Message to be used when no glyph is open.

hTools3.modules.messages.noGlyphSelected = 'No glyph selected.\nPlease select one or more glyphs.\n'¶

Message to be used when no glyph is selected.

hTools3.modules.messages.noLayerSelected = 'No layer selected.\nPlease select one or more layers.\n'¶

Message to be used when no layer is selected.

hTools3.modules.messages.noFontOpen = 'No font open.\nPlease open at least one font.\n'¶

Message to be used when no font is open.

hTools3.modules.messages.onlyOneFont = 'Only one font open.\nPlease open at least one more font.\n'¶

Message to be used when at least two fonts must be open.

hTools3.modules.messages.noFontInFolder = 'Folder does not contain a font.\nPlease add some fonts to the folder, or choose another folder.\n'¶

Message to be used when there are no font files in a given folder.
