=====
proof
=====

Proof tools generate different kinds of proofs for one or more fonts.

Some proofs are created directly from the UFO source fonts, others require generating an OpenType font.

All proof tools use the drawBot module to create PDF documents, and a DrawView to preview the PDF output.
