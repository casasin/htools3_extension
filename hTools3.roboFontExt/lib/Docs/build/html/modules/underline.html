underline¶

addUnderline(sourceGlyph, targetGlyph, distanceY, thicknessY, bufferX, offsetX, clear=False, angle=0)[source]¶

Add an underline stroke with the given parameters to an existing glyph.

Parameters:
  • sourceGlyph (RGlyph) – The input glyph to be underlined.
  • targetGlyph (RGlyph) – The destination glyph where the underlined shape will be drawn.
  • distanceY (int) – Distance to the bottom of the underline shape.
  • thicknessY (int) – Thickness of the underline.
  • bufferX (int) – Amount of units to exceed the glyph margins horizontally.
  • offsetX (int) – Amount of offset around contours to prevent overlapping shapes.
  • clear (bool) – Clear the contents of the target glyph before drawing.
  • angle (float) – The font’s slant angle.
 
sourceGlyph = CurrentGlyph()
targetGlyph = sourceGlyph.getLayer('mask')
angle = CurrentFont().info.italicAngle
addUnderline(sourceGlyph, targetGlyph, 120, 80, 30, 30, clear=True, angle=angle)
addUnderlineFont(font, distanceY, thicknessY, bufferX, offsetX, exceptions=None)[source]¶

Add underline stroke to all glyphs in the font.

Parameters:
  • font (RFont) – The input font with glyph contours to be underlined.
  • distanceY (int) – Distance to the bottom of the underline shape.
  • thicknessY (int) – Thickness of the underline.
  • bufferX (int) – Amount of units to exceed the glyph margins horizontally.
  • offsetX (int) – Amount of offset around contours to prevent overlapping shapes.
  • exceptions (list) – A list of glyph names which should not be underlined.
 
f = OpenFont('path/to/font.ufo', showInterface=False)
addUnderlineFont(f, 120, 80, 30, 30, exceptions=['underscore', 'space'])
f.openInterface()
